## Project Description
This repository contains a group assignment for the Bioinformatics course.  
The aim of this assignment is to demonstrate the use of different programming languages commonly used in bioinformatics.

Each member of the group created a script that prints their personal information including:

1. Full Name  
2. Email Address  
3. Slack Username  
4. Area of Interest in Bioinformatics  

The scripts are stored inside the **scripts folder** in this repository.

---

## Objectives of the Assignment
The objectives of this assignment include:

- To introduce students to programming in bioinformatics
- To demonstrate the use of different programming languages
- To teach students how to collaborate using GitHub
- To understand version control and code sharing
- To develop teamwork skills in computational biology projects

## Workflow

The workflow of this project describes the steps followed by the script to process and store the information of the group members.

Step 1: Define the personal information of the group member including:
- Full Name
- Email Address
- Slack Username
- Area of Interest in Bioinformatics

Step 2: The script prints the information to the screen in an organized format.

Step 3: The script stores the information in a CSV file for structured data storage.

Step 4: The program finishes execution after successfully saving the data.

Workflow Diagram

Start  
↓  
Define Personal Information  
(Name, Email, Slack Username, Area of Interest)  
↓  
Display Information on Screen  
↓  
Save Information to CSV File (bioinformatics_members.csv)  
↓  
End
